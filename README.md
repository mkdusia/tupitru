# $\color{#48d4b1}{\text{Tupitru!}}$
It is a dynamic multiplayer web game that combines fast thinking and analytical skills. One device acts as the game host. Players join the session from their own devices to compete in real-time.

## $\color{#ebb728}{\text{About the Game}}$
The stars of the game are colourful moles. In every round, one of the moles needs to reach a randomly chosen target. Your goal? Help him get there! You must guide the mole to the destination using the minimum number of moves.

## $\color{#f567d8}{\text{How to Play?}}$
* **Host a Game:** Launch the app on one main screen.

* **Join the Party:** Other players connect via their own smartphones/devices.

* **Visit:** [Tupitru!](https://tupitru.tfkls.dev/) and join the game!
