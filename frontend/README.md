# Tupitru Frontend

- **Framework:** React.js
- **Game Engine:** Phaser.js
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** CSS Modules

## Setup

Before running the project, ensure you have the following installed:

- **Node.js**
- **npm** _(usually comes with Node.js)_

### Project Installation & Run

```bash
cd frontend
npm install
npm run dev
```
