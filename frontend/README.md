# Tupitru Frontend

* **Framework:** React.js
* **Game Engine:** Phaser.js  
* **Build Tool:** Vite
* **Language:** TypeScript
* **Styling:** CSS Modules

## Setup

Before running the project, ensure you have the following installed:
* **Node.js** 
* **npm** *(usually comes with Node.js)*

### Project Installation & Run
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
